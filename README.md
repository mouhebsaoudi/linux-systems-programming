# Linux Systems Programming

Hands-on systems programming work completed as part of a university internship at the **Technical University of Munich (TUM)**.  
This repository contains implementation-focused tasks covering core concepts of **Linux systems programming** in **C, C++, and Rust**.

## Overview

The work in this repository focuses on low-level software development and operating system concepts through practical programming assignments.  
The tasks are designed to build a strong understanding of how programs interact with the operating system and how system-level software can be analyzed, debugged, and optimized.

Topics covered include:
- kernel and system calls
- file I/O
- concurrency and synchronization
- memory management
- processes
- sockets and efficient server design
- performance profiling
- virtualization and resource isolation
